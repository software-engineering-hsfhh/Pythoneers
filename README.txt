Um Trump Smasher spielen zu können, musst du folgende Schritte ausführen: 

1. Lade Python herunter (mindestens Python 3.7) z.B. unter folgendem Link: https://www.python.org/downloads/
2. Installiere die Arcade Libary auf deinem Computer
	2.1 Öffne deine Eingabekonsole und gib "pip install arcade" ein
	2.2 Bestätige mit Enter und warte bis die Installation abgeschlossen ist
3. Nun Lade alle Elemtente aus dem folgenden Mainbranch runter  https://github.com/software-engineering-hsfhh/Pythoneers
	und lege sie in einem Ordner ab
4. Öffne in der Konsole den Ordner durch den Befehl "cd Pfad des Ordners"
5. Öffen nun das Spiel durch den Befehl "python3 asteroids.py" oder den Befehl "python asteroids.py"
